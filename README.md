 * - internal

 * Return Type: int, bool, string, object...
 * 
 * Method Name: Verb + Noun
 * 
 * Parameter List:
 * - Parameter Type
 * - Parameter Quantity
 * - Parameter Order
 * - Parameter Keyword: in, out, ref, params
 *
